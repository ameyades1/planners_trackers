🖨️ How to Print

Both trackers are optimized for A4 landscape printing:

Printer Settings:

  • Paper size: A4
  • Orientation: Landscape
  • Margins: 0mm (none) — margins are built into the PDF
  • Scale: 100% (actual size)

💻 Digital Version

Use the HTML files in your web browser:

  • Type habit names in the leftmost column
  • Click checkbox squares to mark daily completions
  • Data persists in browser storage (no account needed)
  • Works on all modern browsers (Chrome, Firefox, Safari, Edge)

⚙️ Customization

Both trackers are fully customizable. Edit the HTML files to:

  • Change colors in the CSS section
  • Modify the number of habits
  • Adjust spacing and fonts
  • Add or remove sections
